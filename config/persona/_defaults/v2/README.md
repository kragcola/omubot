# Persona Defaults v2

These files are proposal-level defaults for Persona Source Importer drafts.

Scope for Part B dry-run:

- `guard.yaml`
- `eval.yaml`
- `trace.yaml`
- `runtime.yaml`
- `state.yaml`
- `thinker.yaml`
- `adapter.yaml`
- `capabilities.yaml`
- `system.yaml`

`system.yaml` mirrors the canonical SystemModule catalog in
`services/system_module/catalog.py`. `modules/` still only gets a placeholder
README until concrete module implementations land.

These defaults are safe to track in git. Real persona drafts, frozen sources,
and pending freezes remain ignored.
