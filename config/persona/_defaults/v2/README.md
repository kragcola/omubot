# Persona Defaults v2

These files are proposal-level defaults for Persona Source Importer drafts.

Scope for the first importer slice:

- `guard.yaml`
- `eval.yaml`
- `trace.yaml`

The remaining v2.1 engineering files (`runtime.yaml`, `capabilities.yaml`,
`adapter.yaml`, `state.yaml`, `thinker.yaml`, `system.yaml`, and
`modules/`) are generated as empty skeletons by the importer until their
runtime consumers are implemented.

These defaults are safe to track in git. Real persona drafts, frozen sources,
and pending freezes remain ignored.
