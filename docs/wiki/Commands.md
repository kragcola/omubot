# 命令

所有命令以 `/` 开头，群聊和私聊均可使用。

## 内置命令

| 命令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/debug [指令]` | — | 管理员 | 调试模式，直接执行工具或查看状态 |
| `/plugins` | `/p`, `/plg`, `/插件` | 管理员 | 列出所有已加载插件 |
| `/version` | — | 所有人 | 查看版本并检查 GitHub 更新 |

## /debug 详解

调试模式下可直接执行工具：

```
/debug 发送表情          # 随机发一张表情包
/debug 发送gif动图表情   # 随机发一张 GIF
/debug 发 stk_cdf94f9b   # 发指定 ID
/debug                   # 显示系统状态摘要
/debug 我的好感度是多少
```

## /version 输出示例

```
Omubot v1.0.4
GitHub: https://github.com/kragcola/omubot
最新版本: v1.0.4（发布于 2026-05-02）
已是最新版本
```
