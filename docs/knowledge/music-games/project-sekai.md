# Project SEKAI / 世界计划

## Project SEKAI / pjsk / 世界计划 / sekai 是什么

Project SEKAI 是一款以 VOCALOID 虚拟歌手与原创角色企划为核心的音游。中文讨论里常见叫法包括“世界计划”“pjsk”“sekai”，有时也会直接简称“啤酒烧烤”。

这款游戏的讨论通常会同时涉及打歌、角色、活动和卡面养成，所以群聊里有人问“pjsk 是什么”，大多是在问这是一款什么类型的音游，以及它和初音未来相关作品的关系。

关键词：Project SEKAI、世界计划、pjsk、sekai、初音未来、VOCALOID

## Project SEKAI 的判定、FC、AP 一般怎么理解

Project SEKAI 里常见的判定讨论和其他音游类似，核心仍然是 Perfect、Great、Good、Bad、Miss 这类时机评价。FC 通常表示 Full Combo，也就是全连；AP 通常表示 All Perfect，也就是整首歌几乎或全部打到最高判定。

玩家在群聊里说“差一 Great AP”“保 FC”“这首歌容易断”时，通常是在讨论稳定度、读谱和手法压力，而不是单纯讨论角色养成。

关键词：判定、FC、AP、Full Combo、All Perfect、Great、断连、稳定度

## Project SEKAI 的编队、属性和技能分主要看什么

讨论 pjsk 编队时，常见会同时提到属性、角色卡面、技能效果和活动加成。简单理解就是：编队不只是看单卡强度，还要看队伍整体属性是否匹配当前活动或目标曲目。

如果群友问“pjsk 编队看什么”“属性加成怎么配”“技能分是什么意思”，通常是在问如何让分数更高，或者怎样更适合活动刷分与冲榜。

关键词：编队、属性、技能分、队伍、刷分、冲榜、活动加成

## Project SEKAI 里的箱活、混活、马拉松、嘉年华 是什么

在玩家讨论里，箱活通常指更偏向某个组合或主题角色群的活动；混活通常指角色组合更混合的活动。马拉松一般是在说常规高频刷歌型活动语境；嘉年华则常被用来指带有阵营、对战或特殊结算感受的活动模式语境。

这些词是玩家社区中的高频说法，具体规则会随着版本和活动设计变化，但在群聊里通常只需要先理解它们分别是在描述“活动归属感”和“活动玩法节奏”。

关键词：箱活、混活、马拉松、嘉年华、活动、刷歌、冲活动
