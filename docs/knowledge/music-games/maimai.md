# maimai / 舞萌

## maimai / 舞萌 / 洗衣机 / maimai DX 是什么

maimai 是一款以圆形框体、外圈操作和街机氛围著称的音游。中文讨论里常见叫法包括“舞萌”“maimai”“洗衣机”，新版语境里也常看到 `maimai DX` 或直接说 `DX`。

如果群友问“maimai 是什么”，通常是在问它为什么和常见下落式音游手感差这么多。最核心的区别就是：它更强调圆盘式交互、外圈动作和街机现场感。

关键词：maimai、舞萌、洗衣机、maimai DX、DX、街机音游

## maimai 的 rating / ra / 推分 一般是什么意思

在 maimai 讨论里，rating 是最常见的实力展示词之一，有时也会被简称成 `ra`。玩家说“推 rating”“灌分”“刷分”，通常是在讨论怎样通过更合适的谱面成绩提升账号的整体评价。

它不等于玩家全部实力，但在交流时是一个非常高频的参考指标，所以有人问“rating 是什么”，一般就是在问这个综合评价数值是怎么被理解和使用的。

关键词：rating、ra、推分、灌分、刷分、成绩评价

## maimai 的 DX 谱、标准谱、段位 常在说什么

玩家讨论 `DX 谱` 时，通常是在区分不同谱面或版本语境；讨论段位时，则更偏向阶段性挑战与实力认证。群聊里说“这首 DX 谱难”“段位过了没有”，本质上都还是在讨论谱面难度和稳定度。

如果 Bot 需要回答这类问题，重点是先识别这属于 maimai 术语，再用“谱面类型”“挑战等级”“成绩目标”这几个维度去解释。

关键词：DX 谱、标准谱、段位、谱面、挑战、稳定度

## maimai 的判定、AP、SSS+ 大致怎么理解

maimai 讨论判定时，常会同时出现 AP、FC、SSS、SSS+ 这类成绩词。它们通常分别对应“更严格的高质量完成度”和“更高等级的分数评价”。

不同玩家在群聊里提这些词时，可能是在讨论手法、准度、读谱，也可能是在讨论某首歌值不值得拿来推分。理解时可以先把它们看成“成绩质量标签”。

关键词：判定、AP、FC、SSS、SSS+、准度、成绩评价
