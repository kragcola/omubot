# 音游基础知识

## 音游 / 节奏游戏 / rhythm game 是什么

音游通常指“音乐游戏”或“节奏游戏”。玩家需要跟随音乐节拍，在正确时机点击、滑动、长按或完成其他操作，以获得更高判定和分数。

不同音游的操作界面和规则差异很大，但常见目标都包括：打准节奏、减少失误、维持连击、提高分数，或者完成某些段位与挑战目标。

关键词：音游、音乐游戏、节奏游戏、rhythm game、打歌

## 判定、连击、FC、AP 是什么

判定就是系统对每一个音符操作时机的评价，常见会出现 Perfect、Great、Good、Bad、Miss 这类等级。判定越高，分数通常越高。

连击就是连续不失误地完成音符。FC 一般指 Full Combo，表示全连，没有断连；AP 一般指 All Perfect，表示整首曲子的判定几乎或全部达到最高等级。不同作品对 AP 的具体叫法和细节可能略有不同，但核心意思都是“比 FC 更严格”。

关键词：判定、连击、FC、Full Combo、AP、All Perfect、Perfect、Great、Miss

## 推分、定数、段位、底力 这些词大致是什么意思

推分通常是指为了让账号分数评价更高，反复挑战合适曲目来提升成绩。定数一般是玩家用来衡量谱面理论难度的参考数值，不同游戏的定数体系不一定互通。

段位通常是官方或社区常见的一种阶段性实力认证；底力更接近玩家对综合读谱、手法、稳定性和硬实力的口语概括。它们都不是所有音游通用的唯一标准，但在讨论强弱时很常见。

关键词：推分、上分、定数、段位、底力、实力、分数评价

## 在群聊里怎么问音游问题更容易命中知识库

如果希望 Bot 更稳定地命中知识库，最好直接把游戏名、缩写和问题关键词一起问出来，例如：

- `@Bot pjsk 的编队主要看什么`
- `@Bot arcaea 的 ptt 是什么`
- `@Bot maimai 的 rating 怎么理解`

当前知识库更擅长命中“标题里出现过、正文里也明确写过”的关键词，所以问题写得越具体，召回通常越稳定。

关键词：怎么提问、知识库命中、音游问答、pjsk、arcaea、maimai
